# Graph Visualizer - Plan
**Timestamp**: 2026-03-05_12-00
**Stack**: React + TypeScript + React Flow

## Goal
A web UI for visualizing graphs where:
- Nodes have a name and arbitrary key-value attributes
- Clicking a node opens a detail panel showing its attributes
- Edges (connections) have a name/label displayed on the connection line
- Users can pan, zoom, and interact with the graph

## Architecture

### Pages / Components
```
App
├── GraphPage                  # main page, holds state
│   ├── GraphCanvas            # wraps ReactFlow, renders nodes/edges
│   │   ├── CustomNode         # node with name label, styled
│   │   └── CustomEdge         # edge with name label
│   └── NodeDetailPanel        # slide-in panel shown on node click
└── Toolbar (optional)         # buttons: add node, add edge, clear
```

### Data Model
```ts
interface NodeData {
  name: string;
  attributes: Record<string, string>;
}

interface EdgeData {
  name: string;
}
```
React Flow's `Node<NodeData>` and `Edge<EdgeData>` types are used throughout.

### State Management
- Local React state (useState) in `GraphPage` — no Redux/Zustand needed for this scope
- Selected node stored in state, drives NodeDetailPanel visibility

## Key Decisions
- **No backend** — data lives in memory (future: could add JSON import/export)
- **React Flow** handles layout, pan/zoom, node/edge rendering
- **Custom node component** for styled name display
- **Custom edge component** for edge label display (or use React Flow's built-in label prop)
- **Vite** as build tool (fastest setup for React + TS)
- **Plain CSS modules** — no Tailwind or CSS-in-JS to keep it simple

## Features (MVP)
1. Display a graph with named nodes and named edges
2. Click node → show detail panel with name + attributes
3. Pan and zoom
4. Hardcoded sample graph as initial data (demonstrates the UI)
5. Add node / add edge via simple form in the UI

## Out of Scope (YAGNI)
- Backend / persistence
- Graph layout algorithms (React Flow handles basic layout)
- Import/export (nice to have, not MVP)
- Authentication
- Multiple graphs
